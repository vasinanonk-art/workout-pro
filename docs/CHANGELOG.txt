# v5.0.1 User Isolation Fix

Critical Fix:
- Firestore workout data is now scoped by Google user UID.
- Team ID is now a group/label, not the primary data owner.
- Users with the same Team ID will no longer see each other's workout logs.

New Firestore path:
- teams/{teamLabel}/users/{userUid}/workouts

Old shared path:
- teams/{teamId}/workouts
- Not deleted.
- Not written to by this build.

Preserved:
- Existing Team ID UI
- Login / Save set
- Dashboard / Log / Calendar / Coach / Program / Guide / Donate
- v5.0 dashboards and export tools
